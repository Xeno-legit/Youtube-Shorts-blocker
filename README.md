# NoShorts: Block YouTube Shorts

A Chrome extension that removes YouTube Shorts from your browsing experience and redirects them to the normal video player.

## Features

-  Automatically redirects Shorts URLs to the regular video player, because you aint slick.
-  Removes Shorts from homepage, sidebar, search results, and channel pages, why else would u download this?
-  Works good enough with youtube being an absolute shi-.
-  Blocks Shorts shelves, tabs, and recommendations

## Installation

1. **Download the extension**
   - Click the green "Code" button above
   - Select "Download ZIP"
   - Unzip the downloaded file to a folder on your computer

2. **Enable Developer Mode in Chrome**
   - Open Chrome and go to `chrome://extensions`
   - Toggle "Developer mode" on (top right corner)

3. **Load the extension**
   - Click "Load unpacked"
   - Select the folder where you unzipped the extension
   - The extension should now appear in your extensions list

4. **Done!**
   - Visit YouTube and enjoy a Shorts-free experience
   - The extension works automatically - no configuration needed

## How It Works

- When you visit a `/shorts/` URL, it instantly redirects to the normal video player
- Continuously monitors the page and removes any Shorts elements as they load
- Uses CSS to hide Shorts buttons, shelves, and recommendations

## Troubleshooting

If Shorts are still appearing:
- Refresh the YouTube page
- Reload the extension from `chrome://extensions`
- Make sure the extension is enabled
